# 🚀 SkillBridge

### Bridge Your Skills to Your Career

SkillBridge is a career & learning platform designed to help graduates and youth discover courses, track their learning progress, and find job opportunities—all in one place.

---

## ✨ Features

- 🔐 **Authentication** – Secure register/login with JWT
- 👤 **User Profiles** – Manage skills, career goals, and bio
- 📚 **Course Management** – Browse, filter, and enroll in courses
- 📊 **Progress Tracking** – Track learning progress (0-100%)
- 📝 **Skill Assessments** – Test your knowledge with quizzes
- 💼 **Job Board** – Browse and apply for jobs/internships
- 📈 **Dashboard** – View stats and recent activity

---

## 🛠️ Tech Stack

### Backend
- [NestJS](https://nestjs.com/) – Progressive Node.js framework
- [Prisma](https://www.prisma.io/) – Type-safe ORM
- [PostgreSQL](https://www.postgresql.org/) – Relational database
- [Supabase](https://supabase.com/) – Database hosting
- [JWT](https://jwt.io/) – Authentication
- [Swagger](https://swagger.io/) – API documentation

### Frontend (Coming Soon)
- [React](https://reactjs.org/) – UI library
- [Tailwind CSS](https://tailwindcss.com/) – Styling
- [Vercel](https://vercel.com/) – Deployment
